# Quick Start Guide

## Get Started in 3 Simple Steps

### Step 1: Run the Application
```bash
python main.py
```

### Step 2: Start Your First Study Session
1. Select option `1` - Start a study session
2. Choose a subject (or create a custom one)
3. Focus on your study for 25 minutes
4. Rate your focus when the timer completes

### Step 3: View Your Progress
- Select option `2` to see your statistics
- Select option `5` to get AI-powered insights about your study habits

## Quick Tips

### For First-Time Users
- Start with the default settings (25-minute work sessions)
- Be honest with your focus ratings to get better AI insights
- Complete at least 5-10 sessions before relying heavily on AI suggestions

### Best Practices
1. **Before Starting**: Clear your workspace and eliminate distractions
2. **During Sessions**: Focus solely on your chosen subject
3. **Take Breaks**: Use break time to rest, not to check social media
4. **Track Honestly**: Accurate data = better AI recommendations
